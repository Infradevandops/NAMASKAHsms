## 🎓 How It Works (Simple Explanation)

### Area Code Retry
1. You request 212 (NYC)
2. System tries to get 212
3. If it gets 713 (Houston) → Cancel and try again
4. Try 917, 646, 347 (other NYC codes)
5. After 3 tries → Accept whatever is available

### Carrier Verification (Tier-Aware)
1. You request Verizon
2. System gets a number
3. Looks up real carrier using Numverify
4. If it's T-Mobile → Cancel and try again
5. After 3 tries → Accept and refund:
   - **PAYG**: Refund $0.30 surcharge
   - **Pro/Custom**: Refund overage cost ($0.30/$0.20)

### VOIP Rejection
1. System gets a number
2. Checks if it's VOIP/landline
3. If yes → Cancel and try again
4. Only accepts mobile numbers
