## 🔄 Maintenance Guidelines

### Going Forward

**When to Archive**:
- Work is complete and deployed
- Issue is resolved and verified
- Audit is finished and acted upon
- Document is outdated or superseded

**How to Archive**:
1. Move to appropriate archives subdirectory
2. Update links in other documents
3. Add entry to INDEX.md if significant
4. Keep original filename for traceability

**What to Keep Active**:
- Current project status
- Setup and configuration guides
- Living documentation (APIs, runbooks)
- Active tasks and roadmaps
- Business logic and policies
