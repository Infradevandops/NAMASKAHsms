## 🔍 When to Archive

Documents should be moved to archives when:

1. **Work is Complete** - Feature implemented and deployed
2. **Issue is Resolved** - Bug fixed and verified
3. **Audit is Finished** - Assessment complete and acted upon
4. **Plan is Executed** - Roadmap items completed
5. **Document is Outdated** - Information no longer current

---

## 📋 Archive Guidelines

### What to Archive
- ✅ Completed implementation summaries
- ✅ Resolved bug reports
- ✅ Finished audits and assessments
- ✅ Executed roadmaps and plans
- ✅ Historical status reports

### What NOT to Archive
- ❌ Active tasks and todos
- ❌ Current project status
- ❌ Living documentation (APIs, guides)
- ❌ Operational runbooks
- ❌ Active roadmaps

### How to Archive
1. Move document to appropriate subdirectory
2. Update links in other documents
3. Add entry to main INDEX.md if significant
4. Keep original filename for traceability
