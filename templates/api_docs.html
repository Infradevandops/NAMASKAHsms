API Documentation

Complete guide to integrating with Namaskah API

Overview

The Namaskah API provides a RESTful interface for SMS verification, email verification, and phone rental services.

Base URL: https://api.namaskah.com/v1

Response Format: JSON

Authentication

All API requests require authentication using an API key. Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY

Get your API key from your account settings.

Endpoints

SMS Verification

POST /sms/send

Send SMS verification code

Rate Limits

API requests are rate limited to prevent abuse:

  • Free tier: 100 requests/hour
  • Pro tier: 1,000 requests/hour
  • Enterprise: Custom limits

Rate limit information is included in response headers:

X-RateLimit-Limit: 1000 X-RateLimit-Remaining: 999 X-RateLimit-Reset: 1700000000
